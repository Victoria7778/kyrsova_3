## Опис проєкту

**Система відстеження настрою та сну** — це веб-застосунок, що дозволяє користувачам:

- вести щоденник емоційного стану
- фіксувати рівень настрою
- відстежувати тривалість сну
- аналізувати історію активності через інтерактивний календар

Мета проєкту — допомогти користувачам краще розуміти свій емоційний стан та зв’язок між якістю сну і настроєм.

---

##  Основні можливості
- Реєстрація та вхід користувачів
- Аутентифікація через **JWT (JSON Web Token)**
- Фіксація рівня настрою (шкала від 1 до 10)
- Вибір типів емоцій
- Введення тривалості сну
- Прив'язка записів до дати
- Перегляд історії сну
- Інтерактивний календар активності
- Перегляд історії записів
- Налаштування особистих даних
- Редагування інформації профілю



###  Стек: MERN
- **MongoDB** — база даних
- **Express.js** — серверний фреймворк
- **React** — клієнтська частина
- **Node.js** — серверна платформа

###  Frontend
- React Router
- Axios

###  Backend
- Mongoose
