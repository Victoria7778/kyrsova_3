## Опис проєкту

**Система відстеження настрою та сну** — це веб-застосунок, що дозволяє користувачам:

- вести щоденник емоційного стану
- фіксувати рівень настрою
- відстежувати тривалість сну
- аналізувати історію активності через інтерактивний календар
- Застосунок забезпечує взаємодію між **користувачем (пацієнтом)** та **фахівцем (психологом)**, дозволяючи відстежувати:



#  Основні можливості

## Для пацієнта

###  Багатовимірний щоденник
- фіксація рівня настрою (**шкала 1–10**)
- вибір типу почуттів
- приватні нотатки

###  Моніторинг фізичного стану
- тривалість сну
- якість сну
- реєстрація фізичних симптомів

### Конфіденційність
- особисті коментарі пацієнта **зашифровані або приховані**
- психолог не має доступу до приватних записів

###  Підключення до фахівця
- зв’язок із психологом через **унікальний ID**

---

## Для психолога

###  Панель керування пацієнтами
- список усіх підключених користувачів
- індикація активності за поточну добу

###  Анонімна аналітика
- графіки динаміки настрою
- без доступу до приватних коментарів
